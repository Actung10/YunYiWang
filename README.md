# 云译网
来自“当然不是草台班子”的一个翻译系统

## 原型
首页：包含了在线翻译、数据记录等功能。其中输入的翻译文本会自动录入首页的AI对话记忆中，方便进行相关信息的检索。
AI对话页面：包含信息检索和文本自动润色的功能。
个性化定制页面：可以根据用户喜好自定义翻译风格、专业术语库等。
